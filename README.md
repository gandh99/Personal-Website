# Personal-Website
My personal website to practice the basics of Web Development. This website was built without any frameworks/bootstrap, only using HTML, CSS and Javascript. It showcases my other projects, work, education and more. It is mobile-friendly and compatible with most popular browsers.

[Curious? Check it out for yourself here.](https://gandh99.github.io/Personal-Website/)
